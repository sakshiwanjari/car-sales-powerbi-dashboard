# Car Sales Power BI Dashboard

This project presents an interactive Power BI dashboard analyzing car sales performance across different companies, regions, and vehicle types.

## Tools Used
- Power BI
- DAX
- Excel / CSV dataset

## Dashboard Insights
- YTD sales performance tracking
- Average car price analysis
- Car sales by body style and color
- Regional dealer sales distribution
- Company-wise sales trends
